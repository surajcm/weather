A basic weather display board based on spring-boot.
</p>

##### Initial Setup
Set the environment variables: 
JAVA_HOME & ACCESS_KEY

##### Build command :
`./gradlew clean idea`

##### To Run the application :
`./gradlew clean bootRun`

#### UI
Please check at http://localhost:8080/
